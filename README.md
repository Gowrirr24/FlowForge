# Traffic-Responsive Road Design System

This project visualizes traffic flow at road junctions based on PCU/hr inputs.
It generates lane requirements, junction type, and animated traffic movement.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Dynamic lane calculation
- Animated traffic visualization
- Slip lanes and traffic islands
- Export design as PNG and JSON
